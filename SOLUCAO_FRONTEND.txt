📋 O PROBLEMA
════════════════════════════════════════════════════════════════════════════

Você vê: "Directory listing for /helpdesk/"
Esperado: Página de login da aplicação

Isto acontece porque python -m http.server não carrega index.html automaticamente.


🔧 SOLUÇÃO 1: USAR PYTHON HTTP SERVER COM DEFAULT (RECOMENDADO)
════════════════════════════════════════════════════════════════════════════

NO TERMINAL 2, CANCELE (Ctrl+C) o servidor atual.

Depois corra ESTE comando em vez:

Windows/Mac/Linux (QUALQUER UM):
  cd frontend
  python -m http.server 5500 --directory .

OU mais simples ainda:

  python -m http.server 5500


DEPOIS:

1. Abra browser: http://localhost:5500/index.html

2. Ou crie um ficheiro que redirecione:

   Abra VS Code
   Clique em: File → New File
   Salve como: .htaccess
   Cole isto:
   
   <IfModule mod_rewrite.c>
     RewriteEngine On
     RewriteBase /
     RewriteRule ^index\.html$ - [L]
     RewriteCond %{REQUEST_FILENAME} !-f
     RewriteCond %{REQUEST_FILENAME} !-d
     RewriteRule . /index.html [L]
   </IfModule>


🔧 SOLUÇÃO 2: USAR OUTRO SERVIDOR HTTP (MELHOR)
════════════════════════════════════════════════════════════════════════════

Instale http-server (Node.js):

1. Instale Node.js: https://nodejs.org/

2. No terminal:
   npm install -g http-server

3. Na pasta frontend:
   cd frontend
   http-server -p 5500 -c-1

4. Abra: http://localhost:5500/


🔧 SOLUÇÃO 3: CRIE UM SCRIPT PYTHON PERSONALIZADO
════════════════════════════════════════════════════════════════════════════

1. Na pasta frontend, crie: server.py

2. Cole isto:

import http.server
import socketserver
import os

os.chdir('.')

class MyHTTPRequestHandler(http.server.SimpleHTTPRequestHandler):
    def do_GET(self):
        if self.path.endswith('/'):
            self.path = self.path + 'index.html'
        return http.server.SimpleHTTPRequestHandler.do_GET(self)

PORT = 5500

Handler = MyHTTPRequestHandler

with socketserver.TCPServer(("", PORT), Handler) as httpd:
    print(f"Server running at http://localhost:{PORT}/")
    print("Press Ctrl+C to stop")
    httpd.serve_forever()

3. Corra:
   python server.py

4. Abra: http://localhost:5500/

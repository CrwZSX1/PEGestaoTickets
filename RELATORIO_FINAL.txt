🚀 COMO COMEÇAR

  1. Backend (terminal 1):
     $ cd backend
     $ python3 -m venv .venv
     $ source .venv/bin/activate
     $ pip install -r requirements.txt
     $ python -m app.seed
     $ uvicorn app.main:app --reload

  2. Frontend (terminal 2):
     $ cd frontend
     $ python -m http.server 5500

  3. Abrir browser:
     http://localhost:5500

  Credenciais:
     admin@helpdesk.local / admin123

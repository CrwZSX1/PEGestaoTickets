🚀 INÍCIO RÁPIDO (5 MINUTOS)
════════════════════════════════════════════════════════════════════════════

1️⃣  Backend (abre terminal 1):

    $ cd helpdesk/backend
    $ python3 -m venv .venv
    $ source .venv/bin/activate     # Windows: .venv\Scripts\activate
    $ pip install -r requirements.txt
    $ python -m app.seed
    $ uvicorn app.main:app --reload

2️⃣  Frontend (abre terminal 2):

    $ cd helpdesk/frontend
    $ python -m http.server 5500

3️⃣  Abre o browser:

    http://localhost:5500

⚠️ NOTAS IMPORTANTES
════════════════════════════════════════════════════════════════════════════

1. Database:
   • SQLite é OK para desenvolvimento
   • Para produção, usa PostgreSQL

2. Email:
   • IMAP está desativado por default (.env IMAP_ENABLED=False)
   • Para testar, instala Mailpit (brew install mailpit)

3. Variáveis de ambiente:
   • Copia .env.example para .env
   • Muda SECRET_KEY em produção!

4. Frontend:
   • Não precisa de build
   • Abre direto no browser com python -m http.server

🆘 TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════

Backend não arranca?
  └─ Verifica se a porta 8000 está livre: lsof -i :8000
  └─ Recria venv: rm -rf .venv; python3 -m venv .venv

Frontend não carrega?
  └─ Verifica se backend está em http://localhost:8000/health
  └─ Limpa cache: Ctrl+Shift+Delete no browser

Login não funciona?
  └─ Verifica if credenciais estão certas
  └─ Limpa localStorage: localStorage.clear() na consola

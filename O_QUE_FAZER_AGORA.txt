  5. URL: https://meu-helpdesk.herokuapp.com

Opção B: Railway (recomendado)
  1. Conta em railway.app
  2. Conectar repositório Git
  3. Configurar variáveis de ambiente
  4. Deploy automático

Opção C: DigitalOcean/AWS (VPS)
  1. Criar droplet/instância
  2. SSH para servidor
  3. Clonar repositório
  4. Instalar dependências
  5. Usar gunicorn + nginx
  6. Configurar certificado SSL

Opção D: Docker (recomendado para tudo)
  1. Criar Dockerfile
  2. $ docker build -t helpdesk .
  3. $ docker run -p 8000:8000 helpdesk
  4. Deploy em qualquer lugar

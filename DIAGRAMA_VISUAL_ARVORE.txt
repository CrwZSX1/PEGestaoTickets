CRIAR TICKET:

  User (TicketListPage.js)
        │
        ├─► preenche form
        │
        ├─► clica "Criar"
        │
        └─► http.post('/tickets', {...})
              │
              └─► backend/routes/tickets.py::create_ticket()
                    │
                    ├─► valida com TicketCreate schema
                    │
                    ├─► cria Ticket model
                    │
                    ├─► calcula SLA (services/sla.py)
                    │
                    ├─► atribui com round-robin
                    │
                    ├─► retorna TicketOut schema
                    │
                    └─► services/email.py envia notificação (background)
                          │
                          └─► template HTML + SMTP


LOGIN:

  User (LoginPage.js)
    │
    ├─► email + password
    │
    └─► POST /auth/login
          │
          └─► routes/auth.py
                │
                ├─► busca User em BD
                │
                ├─► verifica password (auth.py::verify_password)
                │
                ├─► cria JWT token {"sub": user_id, "role": "admin"}
                │
                ├─► retorna token + user info
                │
                └─► frontend armazena em localStorage
                      │
                      └─► todos requests enviam "Authorization: Bearer <token>"


SLA CHECK:

  APScheduler job (a cada 5 min)
    │
    └─► services/sla_checker.py::check_sla_once()
          │
          ├─► busca tickets com sla_deadline < now
          │
          ├─► marca sla_breached = true
          │
          └─► notify_sla_breached() → email
